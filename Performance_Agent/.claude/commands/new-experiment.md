새 실험을 시작한다.

1. `context.md`를 읽어 현재 실험 이력과 마지막 성능을 파악한다.
2. `CLAUDE.md`를 읽어 이미 시도한 기법 목록을 확인한다.
3. 아래를 출력한다:
   - 현재 베이스라인 성능 (수치)
   - 이번 실험에서 변경할 내용 (단 하나)
   - 변경 근거
   - 예상 효과
   - 검증 방법
4. 변경 내용을 코드에 적용한다.
5. 실험 완료 후 결과를 `context.md` 이력 테이블에 다음 형식으로 추가한다:
   `| {다음 ID} | {변경 내용} | {성능} ({이전 대비 증감}) | {비고} |`

단일 변인 원칙: 이 실험에서 바꾸는 것은 정확히 하나여야 한다.
