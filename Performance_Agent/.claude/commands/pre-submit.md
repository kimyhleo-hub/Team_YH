최종 제출 전 체크리스트를 실행한다.

다음 항목을 순서대로 확인하고 각 항목에 ✅ 또는 ❌로 답한다. ❌가 있으면 해결 방법을 함께 제시한다.

**데이터 무결성**
- [ ] 테스트셋 전처리가 학습셋 전처리와 동일한가?
- [ ] 제출 파일에 NaN이 없는가?
- [ ] ID 컬럼 순서가 샘플 제출 파일과 동일한가?
- [ ] 예측값 범위가 요구사항 내인가? (확률: 0~1, 정수 레이블 등)

**모델 및 재현성**
- [ ] 시드가 고정되어 있고 재실행해도 동일 결과인가?
- [ ] 앙상블을 쓴다면 모든 모델의 예측이 최신 버전인가?
- [ ] TTA를 쓴다면 역변환이 올바르게 적용되었는가?

**임계값 및 후처리**
- [ ] 임계값이 OOF 또는 검증셋 기준으로 최적화되었는가? (테스트셋 사용 금지)
- [ ] 앙상블 가중치가 OOF 기준으로 결정되었는가?

**제출 관리**
- [ ] context.md에 이번 제출 내용이 기록되어 있는가?
- [ ] 제출 횟수 제한이 남아 있는가?
- [ ] 이전 최고 제출 대비 성능 변화를 확인했는가?

모든 항목 확인 후 제출 준비 완료 여부를 최종 판단한다.
