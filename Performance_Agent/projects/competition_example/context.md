# Competition Example — 컨텍스트

## 데이터셋

- **크기**: 학습 10,000건 / 테스트 2,500건
- **피처 수**: 30개 (수치형 20, 범주형 10)
- **클래스 분포**: 클래스 0: 7,000건 / 클래스 1: 2,000건 / 클래스 2: 1,000건
- **결측치**: 피처 5개에서 5~15% 결측
- **이상치**: 피처 'amount'에서 상위 1% 이상치 확인됨

## 평가 환경

- **플랫폼**: Kaggle
- **하드웨어**: GPU T4 (15GB)
- **제출 횟수 제한**: 일 5회
- **시드**: 42 고정
