# Research Example — 컨텍스트

## 데이터셋

- **크기**: 학습 50,000장 / 검증 10,000장 / 테스트 10,000장
- **클래스 수**: 10개
- **클래스 분포**: 균등 (각 클래스 5,000장)
- **이미지 크기**: 224×224 (원본 다양, 리사이즈 필요)
- **특이사항**: 클래스 7, 8번에 오클루전 케이스 다수

## 평가 환경

- **하드웨어**: RTX 3090 (24GB)
- **프레임워크**: PyTorch 2.0
- **시드**: 42 고정
- **라이브러리 버전**: torch==2.0.1, torchvision==0.15.2
