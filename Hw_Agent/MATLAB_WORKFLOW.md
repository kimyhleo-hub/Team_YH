# MATLAB Workflow

Last updated: 2026-05-17

## 목적

MATLAB 과제를 `Hw_Agent` 안에서 일관되게 처리하기 위한 최소 운영 문서.

## 입력

- 문제 원문 파일 또는 텍스트
- 과제별 `notes.md`
- 실행 가능 여부
  - MATLAB MCP
  - 로컬 MATLAB 실행 환경

## 표준 절차

1. 문제 요구사항과 제출 산출물을 정리한다.
2. 변수, 파라미터, 출력 물리량을 정의한다.
3. 필요한 식과 비교 기준을 정리한다.
4. `src/solve.m`를 작성한다.
5. MATLAB 실행이 가능하면 실행하고 그래프를 확인한다.
6. 실행 결과를 `solution.md`에 반영한다.
7. 실행이 불가능하면 `실행 미검증`으로 표시하고 수동 실행 방법을 남긴다.

## 산출물

- `{subject}_{assignment}/src/solve.m`
- `{subject}_{assignment}/solution.md`
- 필요시 `{subject}_{assignment}/figures/*`

## 검증 체크

- 문제에서 요구한 모든 그래프가 생성되었는가
- 축, 범례, 제목이 명확한가
- 식과 실제 플롯 물리량이 일치하는가
- 해석 문장이 그래프와 분리되어 있는가
- 실행 여부가 문서에 명시되어 있는가

## 금지

- 실행하지 않은 그래프나 수치를 실제 결과처럼 적지 않는다.
- 출력 물리량이 다른 문제에 식을 그대로 재사용하지 않는다.
