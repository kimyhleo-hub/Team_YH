# Vibration_Hw Notes

Last updated: 2026-05-17

## Scope
- 이 문서는 `Vibration_Hw` 과제에만 적용되는 전용 메모다.
- 공통 작업 규칙은 상위 `AGENT.md`를 따른다.

## Assignment Targets
- 문제 1: 조화 외력 `f(t) = F sin(\omega t)`에 대한 질량 변위 `x`의 응답 비교
- 문제 2: 바닥 가진 `y(t) = Y sin(\omega t)`에 대한 전달력 `F_T` 비교
- 제출물: 식, MATLAB 명령문, 출력 그래프, 파라미터 영향 서술

## Variable Conventions
- `m`: 질량
- `c`: 감쇠계수
- `k`: 스프링 상수
- `\omega`: 입력 주파수
- 문제 1의 그래프: x축 `\omega`, y축 `x`
- 문제 2의 그래프: x축 `\omega`, y축 `F_T`

## Comparison Rules
- 한 번에 하나의 파라미터만 변화시키고 나머지 둘은 고정한다.
- 최소 3개 이상의 비교값을 사용해 경향이 보이게 한다.
- `m` 변화, `c` 변화, `k` 변화는 별도 그래프로 분리한다.
- 문제 1과 문제 2는 같은 방식으로 정리하되, 응답 물리량 정의를 다시 적는다.

## MATLAB Output Rules
- 그래프마다 제목, 축 라벨, 범례를 넣는다.
- 고정값과 변화값은 코드 또는 캡션에서 바로 보이게 적는다.
- 가능하면 한 문제당 서브플롯 또는 일관된 figure 구성으로 정리한다.
- 실행 불가 시 그래프 결과를 추정해서 적지 않는다.

## Solution Writing Rules
- 식을 먼저 쓰고, 그 식이 어떤 물리량을 나타내는지 바로 설명한다.
- 그래프 제시와 영향 해석을 분리해서 쓴다.
- 영향 해석에는 공진 부근 변화, 진폭 변화, 감쇠 효과를 우선 서술한다.
- 파라미터 값이 임의 선정이라면 "타당한 범위에서 선정"했음을 적고 값을 명시한다.

## Validation Hints
- 단위 일관성을 먼저 본다.
- `\omega` 범위가 너무 좁아 공진 경향이 안 보이지 않는지 확인한다.
- `m`, `c`, `k`를 바꿀 때 y축 물리량 정의가 바뀌지 않았는지 확인한다.
- 문제 2는 문제 1의 식을 그대로 복붙하지 말고 전달력 기준으로 다시 정의한다.
